# Surgical Data Management System (Node.js, Express.js & Next.js)

## Project Overview
The **Surgical Data Management System** is a web-based platform designed to digitally store and manage surgical information for hospitals and clinics. It enables efficient handling of patient data, surgeon information, operation schedules, and post-operation reports.
